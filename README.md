<h1>Y-TOT</h1>

<h2>Application</h2>
<p>
  This application was made with two main repo:
  <ul type ="circle">
    <li><a href = "https://github.com/yt-dlp/yt-dlp"> yt-dlp</a></li>
    <li><a href = "https://github.com/FFmpeg/FFmpeg">FFmpeg</a></li>
  </ul>
    Its main goal is to download high quality youtube video without any hassle.
    </br></br>
    This application will automaticlly download and install yt-dlp and FFmpeg in the first run, 
    and there is a consol in the bottom of the application for knowing whats happening in the background.
</p>

<h2>RUN</h2>
<p>
  
### Clone the repository
```bash
git clone https://github.com/Nolin1/Y-ToT.git
cd Y-TOT
```
### Remove Git history (optional)
```bash
rmdir /s /q .git   # Windows
```

### Run the Application
```bash
cd Application  #Go to the MainApp.py Folder
```
```bash
python MainApp.py   #Run the Application
```
</p>
